# 砚台 · 郑超个人网站

国风风格的个人作品站，包含简历的全部板块，并支持后台无关的「编辑模式」。

## 启动

```bash
# 在项目根目录
python3 -m http.server 5566
# 浏览器打开 http://localhost:5566
```

也可以直接双击 `index.html` 打开（部分功能如本地媒体上传需要 http 服务器）。

## 文件结构

```
geren/

│   ├── main.css       # 主题/导航/Hero/通用

└── js/

    ├── store.js       # localStorage 数据层

    ├── portfolio.js   # 作品集：上传/编辑/详情
    ├── edit.js        # 编辑模式（文本/列表项 增删改）
    └── main.js        # 入口、滚动动画、视差
```

## 编辑使用说明

1. 点击右下角 **「编辑」** 按钮进入编辑模式（再次点击退出）。
2. 编辑模式下：
   - 标题、引言、联系方式等文本：**点击直接修改**
   - 头像图片：**点击可上传本地图片或填写图片 URL**
   - 列表项（个人优势、技能、工具、履历、项目、教育）：
     - 点击单条 → 弹窗修改
     - **Shift + 点击** → 删除
     - 点击下方「＋」按钮 → 添加新项
3. 作品集板块：
   - 直接点击右上角「**添加作品**」按钮（任何模式下都能用）
   - 编辑模式下，每个作品卡片显示 ✎ 编辑 / × 删除
   - 上传支持图片和视频，可拖放，可多选
4. **重置默认**：双击右下角「编辑」按钮可恢复简历默认数据。

## 数据存储

- 所有内容保存在浏览器 `localStorage`，刷新/重启后保留
- 媒体文件以 base64 形式保存（建议单个文件 < 8MB）
- 想跨设备保留：在浏览器设置中导出 localStorage，或后续接入云存储
